# 밸런스 게임 웹사이트 (Balance Game Website)

## 프로젝트 개요
이 프로젝트는 사용자가 두 가지 선택지 중 하나를 고르는 "밸런스 게임"을 플레이하고, 각 주제에 대해 다른 사람들과 댓글로 소통할 수 있는 대화형 웹사이트입니다. https://bellruns.com/ 사이트를 참고하여, 고품질의 시각적 디자인과 사용자 경험을 제공하는 것을 목표로 합니다.

## 스타일, 디자인 및 기능

*   **디자인 철학**: 모던하고 جرأة(Bold)한 디자인을 추구합니다. 생생한 색상 팔레트, 풍부한 그림자 효과, 그리고 명확한 타이포그래피를 사용하여 사용자의 시선을 사로잡고 즐거운 경험을 제공합니다.
*   **레이아웃**: 중앙 집중식의 카드형 UI와 그 아래에 댓글 섹션을 배치하여 게임과 토론을 자연스럽게 연결합니다.
*   **반응형 디자인**: 모든 디바이스(모바일, 태블릿, 데스크톱)에서 완벽하게 작동하도록 반응형으로 설계됩니다.
*   **색상 팔레트**:
    *   배경: 부드러운 질감을 가진 연한 회색 (`#f0f2f5`에 노이즈 텍스처 추가)
    *   기본 텍스트: `#333`
    *   주요 색상 (타이틀, 강조): 생동감 있는 보라색 계열 (`#8A2BE2`)
    *   선택지 카드: 흰색 (`#FFFFFF`)
    *   카드 그림자: 깊이감을 주는 다중 레이어 그림자
    *   선택 효과: 선택된 카드는 부드러운 녹색 (`#28a745`) 테두리와 그림자로 강조
*   **폰트**: 가독성이 높고 현대적인 'Pretendard' 웹 폰트를 사용할 것입니다. (CDN을 통해 임포트)
*   **인터랙티브 요소**:
    *   선택지 카드는 마우스를 올리면 살짝 떠오르는 효과를 가집니다.
    *   선택 후에는 통계(%)가 부드럽게 나타나는 애니메이션이 적용됩니다.
    *   선택을 해야만 댓글 기능이 활성화됩니다.

### 구현될 기능
1.  **질문 및 선택지 표시**: 중앙에 현재 질문과 두 개의 선택지 카드를 표시합니다.
2.  **선택 기능**: 사용자가 두 카드 중 하나를 클릭하여 선택할 수 있습니다.
3.  **결과 표시**: 선택이 완료되면, 각 선택지의 백분율 통계가 표시됩니다.
4.  **댓글 기능 (선택 후 활성화)**: 사용자가 특정 선택지를 고르면, 해당 선택지를 고른 다른 사람들과 의견을 나눌 수 있는 댓글 창이 나타납니다. 각 선택지별로 댓글이 따로 관리되며, **댓글 작성 시 닉네임 옆에 자신이 선택한 항목이 괄호와 함께 표시됩니다.**
5.  **다음 질문 이동**: '다음' 버튼을 통해 다음 문제로 넘어갑니다.

### 2단계: 핵심 게임 로직 및 댓글 기능 수정 (JavaScript)
1.  **`main.js` 수정**:
    *   게임 데이터 구조를 선택지별(`A`, `B`)로 댓글을 저장하도록 변경합니다.
    *   `loadQuestion` 함수가 호출될 때 댓글 섹션을 숨기는 로직을 추가합니다.
    *   선택지 카드 클릭 시, 해당 선택지에 맞는 댓글 창을 보여주고 댓글 목록을 불러오는 로직을 구현합니다.

    *   **댓글 제출 시, 사용자가 선택한 선택지의 텍스트를 댓글 데이터와 함께 저장합니다.**
    *   **댓글을 화면에 표시할 때, 닉네임 옆에 저장된 선택 항목 텍스트를 괄호와 함께 출력합니다.**
